Cliquez sur ▶ pour écouter chaque adjectif. Décrire quelqu'un, c'est pouvoir parler des gens autour de vous : taille, corpulence, cheveux, yeux, âge.
1
ADJECTIF 1/8
kbir / sghir
كبير / صغير — Grand / Petit (taille ou âge)
Les deux adjectifs les plus polyvalents du darija. Accord féminin : kbira / sghira.
2
ADJECTIF 2/8
twil / qsir
طويل / قصير — Grand (de taille) / Court
twil décrit une grande stature ou longueur ; qsir décrit quelque chose de petit ou court.
3
ADJECTIF 3/8
smin / rfif
سمين / رفيف — Gros / Mince
Adjectifs utilisés librement au Maroc pour décrire la corpulence, de façon généralement neutre.
4
ADJECTIF 4/8
sh3r khal / sh3r byd
شعر كحل / شعر بيض — Cheveux noirs / Cheveux blancs
sh3r = cheveux ; khal = noir profond ; byd = blanc.
5
ADJECTIF 5/8
sh3r twil / sh3r qsir
شعر طويل / شعر قصير — Cheveux longs / Cheveux courts
Combinaison de « cheveux » avec les adjectifs twil et qsir déjà appris.
6
ADJECTIF 6/8
3yunu khudr / khal / zurq
عيونه خضر / كحل / زرق — Ses yeux verts / noirs / bleus
Au pluriel, les adjectifs de couleur changent de forme pour s'accorder avec « yeux ».
7
ADJECTIF 8/8
zwin / mzyan
زوين / مزيان — Beau / Bien — Super
Deux des mots les plus utilisés en darija pour complimenter ou approuver.
